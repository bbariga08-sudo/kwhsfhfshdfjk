# 💰 Mans Budžeta Plānotājs

## 📝 Apraksts
"Mans Budžeta Plānotājs" ir vienkārša lietotne, kas palīdz lietotājiem sekot līdzi saviem ikdienas izdevumiem un ienākumiem. Tā ir paredzēta studentiem un ikvienam, kurš vēlas labāk pārvaldīt savas finanses un izvairīties no liekiem tēriņiem mēneša beigās.

`git clone https://github.com/lietotajvards/projekta-nosaukums.git`
2. Atveriet mapi savā koda redaktorā (piemēram, VS Code).
3. Terminālī ierakstiet `python main.py` (vai citu palaišanas komandu).

## 🖥️ Lietošana
Pēc programmas palaišanas parādīsies galvenā izvēlne.
**Piemērs:**
- Ievadiet summu: `5.50`
- Izvēlieties kategoriju: `Pusdienas`
- Programma automātiski atņems šo summu no jūsu kopējā mēneša budžeta.

## ✨ Funkcijas
* **Ienākumu/Izdevumu uzskaite** – iespēja pievienot katru darījumu.
* **Kategoriju izveide** – sagrupējiet tēriņus (Pārtika, Izklaide, Transports).
* **Mēneša atskaite** – vizuāls kopsavilkums par to, kur palikusi nauda.

## 📖 Detalizēta instrukcija: Kā pievienot jaunu izdevumu?
Sekojiet šiem soļiem:
1. Galvenajā izvēlnē nospiediet pogu **"Pievienot izdevumu"**.
2. Ievadiet tēriņa summu skaitliskā formātā (piemēram, 10.00).
3. Izvēlieties datumu no kalendāra.
4. Nospiediet pogu **"Saglabāt"**.
5. Jūs redzēsiet apstiprinājumu un jaunais atlikums tiks pārrēķināts.
